// Per-User Services (_5b055)

The _5b055 suffix services throw "parameter incorrect" in services.msc. The script fixes them via registry directly and also disables their templates so they don't respawn on reboot.

// Print Spooler

Disabled in the script. If you ever need to print, re-enable it temporarily. Comment it out of the .ps1 if you print regularly.

// Bluetooth

All BT services are disabled. Comment those lines out if you use a Bluetooth mouse, headset, or controller.

// Windows Update

Set to Manual — not fully disabled. Fully killing WU causes Windows self-repair loops. Manual means it only runs when you trigger it.

// vgc (Vanguard)

Valorant anti-cheat. Not touched by the script. Disabling it means you can't launch Valorant. Leave it.

// Windows Search

Already disabled on your build. Everything by Voidtools covers search — no need for the Windows indexer running in the background.
